Memory

有段時間特別好奇記憶是什麽。

依稀記得看過的某本書上的某位哲人曾説過。

記憶就是火燒在蠟上的景象

火，是什麽。蠟，又是什麽。

這個答案顯然太過抽象，像我這樣笨拙的人是理解不了的。我衹好求助於Wiki，

記憶是神经系统存储过往经验的能力，关于记忆的研究屬於心理學或腦部科學的範疇。記憶代表著一個人對過去活動、感受、經驗的印象累積，有相當多種分類，主要因環境、時間和知覺來分。

看了這個答案，我反而更加疑惑了。如果説記憶是過往的經驗，那麽在我腦海中個偶爾閃過的景象又是怎麽回事呢。

後來有人告訴我，那叫做Déjà vu，是因爲大腦「打結」所產生的幻感。

也就是説一切都是假的，是癔病，是幻覺。

啊，是這樣嗎，所以你們才會一個接一個的消失嗎。

「人總要活得現實一點」

Chip

好的，我明白了….個鬼啊！

不行啊！

不對啊！

絕對有哪裏出了問題啊！

我知道的，那份心意，那種溫柔，還有冬季時環繞在我身邊的她的溫度，都無法作假。

「那就一定是這個世界出了問題」

當我意識到的時候，記憶，在飛速的消退。

被無法抗拒的力量施加著，黑色的巨獸在我腦海中肆虐。

肆意玩弄著，粉碎著的，那是我的記憶——我最寶貴的東西。是從一無所有中誕生的我，能夠保佑的唯一。

就連這樣也不放過我嗎。

就連這樣也要抛棄我嗎。

明明是好不容易才走到這裏的，卻不能與他人擁有相同的權力，這就叫做世界嗎。

如果，如果偏要這樣做的話，那我就把它們——被撕裂開來的記憶全部找回來。然後，用比你還快的速度拼凑起來。

即便無法完全復原，也絕不想讓他們被遺忘。

此刻，面前這些閃著小小光芒的碎片，就是我的人生啊。

どうだろうね。明日になったらすっかり忘れちゃってたりして
